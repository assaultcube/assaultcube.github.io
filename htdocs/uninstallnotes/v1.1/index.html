Version 1.1 Uninstall Notes

Thanks for playing AssaultCube. We hope you had a good time.

AssaultCube stores user-specific data like configuration and additional content in a separate directory.
Although you've removed AssaultCube from your system, this separate directory may still exist. If you want to ensure all user-specific data is removed as well, you should delete the AssaultCube folder that is located in your My Documents folder.
On Linux this folder is located in ~/.assaultcube_v1.1

If you are uninstalling due to issues running AssaultCube, you may want to check for the answer in the official documentation.
If you still can't find the answer you need, search the wiki, and the forums, and if you still can't find the answer, post a thread on the forum explaining your situation.

We would appreciate if you write us a bug report if there is a fault preventing you from running the game.

You may also try IRC for a real-time response, but be patient (at least 30mins), as people "idle" in IRC, and may take time to respond.
